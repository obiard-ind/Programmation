# Le type 'Booléen' : bool

**Note** : Le type booléen permet de représenter le résultat d'une une *expression booléenne* ; à savoir, une expression dont la valeur est soit : `True` (vraie) ou `False` (fausse).  Les *expressions booléennes* sont utilisées pour exprimer des *conditions*.
## Ensemble des valeurs :
{`False`, `True`}

**Rem** : il y a équivalence entre 
- la valeur `False` et la valeur `0`
- la valeur `True` et la valeur `1`

De ce fait, l'ensemble des valeurs pourra également être noté `{0,1}`  
## Opérations définies sur ces valeurs :
### Les opérations 'logiques'
Opérateurs logiques : `and`, `or`, `not`

**Rem** : du fait que les booléens sont représentés par les entiers {`0`, `1`}; toutes les opérations applicables aux entiers sont également applicables aux booléens; et notamment :
- Les opérations arithmétiques,
- Les opération relationnelles,
- etc...
### Les opérations 'arithmétiques'
Opérateurs arithmétiques : `+`, `-`, `*`, `/`, `//`, `%`

### Les opérations 'relationnelles'
Opérateurs relationnels : `<`,`<=`, `>`, `>=`
