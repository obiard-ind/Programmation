# Le type 'Integer' : int

Le type Integer représente les nombres entiers.
## Ensemble des valeurs :
L'ensemble des nombres entiers : ℤ 
Exemples : -7896, 123456, 0 , -7, 15,...
## Opérations définies sur ces valeurs :
### ~~Les opérations 'logiques'~~
Opérateurs logiques : `and`, `or`, `not`

**Rem** : les opérations logiques fonctionnent également sur les entiers; mais sont fort peu utilisées dans la pratique.  Elles sont mentionnées dans ce cours, par soucis de complétude; et ne sont pas à connaître pour les types autres que 'booléens'.
### Les opérations 'arithmétiques'
Opérateurs arithmétiques : `+`, `-`, `*`, `/`, `//`, `%`

### Les opérations 'relationnelles'
Opérateurs relationnels : `<`,`<=`, `>`, `>=`
